# Text to Audio

## About

#### Conversor de texto para audio pt-br

## Install 

#### npm install text-audio-pt-br --save 

### In your code 


```
const speaker = require('text-to-audio');

speaker.read("Eu estou fazendo um teste");
//your mp3 audio will to be in ./yourtext.mp3

```
